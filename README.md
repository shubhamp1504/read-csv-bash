# read-csv-bash
This project demonstrates how to read and process a CSV file in **Linux Bash** using the **IFS (Internal Field Separator)** concept.


## Features
- Reads CSV file line by line
- Uses comma (,) as delimiter
- Skips header row
- Beginner-friendly & DevOps useful

## Technologies Used
- Linux
- Bash Scripting
